Runrift

Runrift ist ein Unity Projekt, welches ich als Abschluss Projekt in der Sek gemacht habe. Man spielt eine blaue Person und muss so weit wie möglich laufen, dabei wird das Level zufällig generiert. Man läuft automatisch und muss mittels der Sprungtaste Hindernissen ausweichen und Tokens einsammeln.

Snake+

Snake+ ist ein Snake Spiel welches ich mit Python umgesetzt habe. In den Optionen kann man die Spielfeldgrösse, die Anzahl der Äpfel und die Geschwindigkeit anpassen, zudem kann man noch Erweiterungen aktivieren, wie Invertierte Steuerung, oder das sich die Schlange nach jedem Apfel dreht.

PacMan

PacMan ist einfach PacMan welches ich mit einem Kollegen in JavaFX versucht habe zu realisieren. Es ist nicht Perfekt, aber es tut was es tuen soll.

ItemManagement

Mein erstes Fullstack Projekt war eine Software zur Verwaltung von Items. Mann kann damit verschiedene Items erfassen, mit Bild, Beschreibung, Lagerort, Tags und noch vielem Mehr. Mann kann die Items suchen, Filtern, bearbeiten und löschen. Umgesetzt mit MySQL, Java Springboot und Vanilla HTML, CSS, JavaScript.

Echo

Echo ist eine App, die sehr an YouTube inspiriert ist. Zusammen mit einem Kollegen habe ich eine React Native App entwickelt, in welcher man Videos erstellen und schauen kann. Zudem kann man Videos oder Kanäle suchen und Kommentare schreiben.

Famegame

Famegame ist eine Webseite auf der man Snake+, PacMan und eine Todo Liste online spielen/verwenden kann. Die Scores werden in einer Datenbank gespeichert und auf einem Leaderboard angezeigt. Ein Fullstack Projekt, welches in Zukunft online gehen wird.
