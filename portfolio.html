Runrift

Runrift ist ein Unity Projekt, welches ich als Abschluss-Projekt in der Sek gemacht habe. Mit einer blauen Figur läuft man so weit wie möglich, dabei wird das Level zufällig generiert. Die Figur läuft automatisch und man muss mittels der Sprungtaste Hindernissen ausweichen und Tokens einsammeln.

Snake+

Snake+ ist ein Snakespiel, welches ich mit Python umgesetzt habe. In den Optionen werden die Spielfeldgrösse, die Anzahl der Äpfel und die Geschwindigkeit angepasst, zudem können noch Erweiterungen aktiviert werden, wie invertierte Steuerung oder dass sich die Schlange nach jedem Apfel dreht.

PacMan

PacMan ist einfach PacMan, welches ich versucht habe, mit einem Kollegen in JavaFX zu realisieren. Es ist nicht perfekt, aber es tut, was davon erwartet wird.

ItemManagement

Mein erstes Fullstack-Projekt war eine Software zur Verwaltung von Items. Damit können verschiedene Objekte mit Bild, Beschreibung, Lagerort, Tags und weiteren Attributen erfasst werden. Die Items lassen sich suchen, filtern, bearbeiten und löschen. Umgesetzt habe ich das Projekt mit einer MySQL-Datenbank, einem Java Springboot Backend und einem Frontend aus Vanilla HTML, CSS und JavaScript.

Echo

Echo ist eine App, die sehr von YouTube inspiriert ist. Zusammen mit einem anderen Kollegen habe ich eine React Native App entwickelt, in welcher man Videos erstellen und anschauen kann. Zudem besteht die Möglichkeit, Videos oder Kanäle zu suchen und Kommentare zu schreiben.

Famegame

Famegame ist eine Webseite auf der Snake+, PacMan und eine Todo-Liste online gespielt/verwendet werden kann. Die Scores aus den Spielen werden in einer Datenbank gespeichert und auf einem Leaderboard angezeigt. Dies ist auch ein Fullstack-Projekt, welches sogar in Zukunft online gehen wird.
